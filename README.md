# MiniMallManager

> An vue backstage management system

#### Build Setup

``` bash
# install dependencies
npm install
or
yarn

# serve with hot reload at localhost:9080
npm run serve
or
yarn run serve

# build electron application for production
npm run build
or
yarn run build
```
